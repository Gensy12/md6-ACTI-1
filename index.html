Investiga en diferentes sitios web y describe cada tema:

  1. Definición de CSS.

    2. CSS (Cascading Style Sheets) es un lenguaje de estilo utilizado para describir la presentación de un documento escrito en HTML o XML. CSS permite a los desarrolladores y diseñadores web separar el contenido del documento (HTML) de su diseño visual, facilitando la creación de interfaces atractivas y responsivas.

  2. Funcionamiento Básico de CSS.

    3. CSS funciona aplicando estilos a los elementos HTML a través de selectores y reglas. Los estilos se pueden aplicar a nivel de página, de forma interna en el documento HTML o de forma externa mediante archivos CSS. Los navegadores interpretan estas reglas para renderizar el contenido en la forma deseada.

  3. Incluir CSS en un archivo HTML.

    4. Hay tres formas principales de incluir CSS en un documento HTML: Inline: Usando el atributo style directamente en las etiquetas HTML. Interno: Dentro de una etiqueta style en la sección head del HTML. Externo: Enlazando un archivo CSS externo usando la etiqueta link también en el head.

  4. Definición de una regla CSS.

    5. Una regla CSS define cómo se debe mostrar un elemento HTML. Consiste en un selector que apunta a uno o más elementos y un bloque de declaración que contiene una o más declaraciones de estilo.

  5. Elemento de una regla CSS.
    1. Regla

      2. Regla: Conjunto completo que incluye un selector y un bloque de declaración.

    2. selector

      3. Selector: Parte de la regla que indica a qué elemento(s) HTML se aplicará el estilo (por ejemplo, p, .clase, #id).

    3. Bloque de declaración

      4. Bloque de declaración: Encerrado entre llaves {}, contiene una o más declaraciones de estilo.

    4. Propiedad

      5. Propiedad: Atributo del estilo que se quiere modificar (por ejemplo, color, font-size).

    5. Valor

      6. Valor: Valor asignado a una propiedad (por ejemplo, red, 16px).

  6. Medios CSS.

    7. Los medios CSS (o media queries) son una técnica que permite aplicar diferentes estilos CSS en función de características del dispositivo, como el ancho de la pantalla, la orientación o la resolución. Esto permite diseñar sitios web responsivos que se adaptan a diferentes dispositivos.

  7. Comentarios CSS.

    8. Los comentarios en CSS se utilizan para insertar notas o explicaciones en el código, sin que afecten la presentación. Se escriben entre /* y */.

  8. Sintaxis de la definición de las propiedades CSS.

    9. La sintaxis de CSS se basa en una estructura que consiste en tres componentes principales: el selector, el bloque de declaración, y dentro de este último, las declaraciones de propiedades.
